# Go-to-market

## Каналы и распределение бюджета

| Канал | CAC | LTV/CAC | Доля бюджета |
|---|---|---|---|
| **Контент-маркетинг** (статьи, SEO) | 400 ₽ (целевой) | 13.8 | **70%** |
| **Таргет** в VK/Telegram | 1 200 ₽ (по тестам) | 4.6 | 30% |
| Партнёрки с площадками для самозанятых | TBD | — | — |

Источник цифр: [[../articles/unit-economics-v1]].

## Логика контент-приоритета

Из [[../articles/meeting-2026-04-15-strategy]] — несколько причин выбрать самозанятых на старте именно из go-to-market соображений:

- Контент по теме самозанятых **дешевле в производстве**
- Готовая аудитория в **нишевых ТГ-каналах** для фрилансеров
- Контент-канал с CAC 400 ₽ даёт payback ~1.5 мес

## Контент-план

Лена готовит контент-план на 3 месяца под самозанятых.
**Дедлайн:** 2026-04-25 ([[../articles/meeting-2026-04-15-strategy]]).

## Партнёрки — кандидаты

Площадки для самозанятых. Конкретные партнёры пока не названы. Возможные направления (предположение, не из исходников):
- Платформы для фрилансеров
- Сервисы для самозанятых (учёт доходов, налоги)
- Профессиональные сообщества по нишам

## Что не работает (по [[../articles/custdev-anna-designer]])

- Регистрация через ЕСИА/Госуслуги — красный флаг для аудитории
- Звонки от менеджеров — отталкивают
- Длинные онбординги — отталкивают

→ **Воронка должна быть максимально короткой:** загрузил договор → увидел ценность → подписался.

## Защита от смежников (на будущее)

Из [[../articles/competitor-pravoved]]: бухгалтерские сервисы (Контур, Моё дело) — потенциальная угроза, потому что у них уже есть наша ЦА. Митигация — **API для бухгалтерских сервисов** в скоупе v2.0 ([[mvp]]). Превращаем потенциального конкурента в канал.
